# 项目简介 python574

本项目是一个基于协同过滤算法的电影推荐系统，使用 Python 语言开发，集成了多种功能模块，旨在为用户提供个性化电影推荐服务。

## 技术栈

### 后端技术

- Python
- Django 框架

### 前端技术

- jQuery
- Bootstrap
- HTML

### 数据库

- MySQL 5.7.33

### 开发工具

- PyCharm

## 功能模块

### 前台管理

- 电影信息浏览
- 个人中心
- 电影搜索
- 电影收藏

### 后台管理

- 个人中心
- 电影分类管理
- 用户管理
- 系统管理

## 系统角色

- 用户：浏览电影信息，进行电影搜索、收藏，管理个人中心
- 管理员：管理电影分类、用户及系统设置

## 运行环境

- MySQL 5.7.33 或兼容版本
